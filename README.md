# Daybreak — Support &amp; Privacy

Public support and privacy pages for the **Daybreak** iPhone app, hosted with GitHub Pages.

- **Support:** [https://jaxonarnold.github.io/daybreak-support/](https://jaxonarnold.github.io/daybreak-support/)
- **Privacy Policy:** [https://jaxonarnold.github.io/daybreak-support/privacy.html](https://jaxonarnold.github.io/daybreak-support/privacy.html)

These are the URLs used in the App Store Connect listing (Support URL and Privacy Policy URL).

## Editing

Both pages are plain, self-contained HTML files (`index.html` and `privacy.html`). Edit them directly and push; GitHub Pages redeploys automatically within a minute or so.
